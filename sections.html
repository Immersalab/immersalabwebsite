Synthetic Data for AI/Robotics

Deterministic datasets for machine learning. Generate perfect training data with controlled randomization. Glint3D's deterministic pipeline ensures reproducible datasets with pixel-perfect labels‚Äîaccelerating ML development and improving model evaluation.

$ python generate_dataset.py

generator = DatasetGenerator(
  scene_template='warehouse_environment.json',
  object_library='objects/',
  randomization_config={
    'lighting': {'intensity': [0.5, 2.0]},
    'camera': {'height': [1.0, 2.5]},
    'objects': {'count': [1, 5]}
  }
)

dataset = generator.generate(
  count=10000,
  outputs=['rgb', 'depth', 'segmentation']
)
