Desktop installers

Signed builds with GPU drivers tuned for workstation and studio deployment.

Windows 11 / 10

Installer with DirectX + Vulkan backends, optional CUDA extensions, and auto-update service.

  • Supports NVIDIA, AMD, Intel ARC.
  • Installs background orchestrator for job queuing.
  • Includes Visual Studio redistributables.

macOS (Apple Silicon)

Universal build optimized for M-series GPUs with Metal acceleration and offline RayKit.

  • macOS 13 Ventura or newer.
  • Native arm64 binary; Intel fallback via Rosetta.
  • Ships with CLI and automation scripts.
Download .dmg

Linux (Ubuntu / Fedora)

AppImage plus shell scripts for systemd services, focused on render farm orchestration.

  • GLIBC 2.31+, kernel 5.15+.
  • Supports NVIDIA and Mesa-based drivers.
  • Includes Docker compose stack and CLI.
Download AppImage

CLI & Docker

Roll Glint3D into CI pipelines, headless render farms, and AI-driven workflows.

  • CLI shipped as `glint` for Windows, macOS, and Linux.
  • Docker images include GPU drivers, scheduler, and observability stack.
  • API tokens with granular scopes for automation agents.
Pull Docker image Read CLI docs 
$ curl -L https://dl.glint3d.dev/install.sh | sh
$ glint login --token xxxx-xxxx
$ glint render --scene city.glb --farm cluster-west

System requirements

  • Quad-core CPU or better, 16 GB RAM minimum.
  • Dedicated GPU with 6 GB VRAM (RTX 2060+/RX 5600 XT+/M-series).
  • 20 GB free storage for caches and AI models.
