Installation & Setup

Get up and running with Glint3D in minutes

Windows Installation

  1. Download the Windows installer (glint3d-windows-x64.zip)
  2. Extract the ZIP file to your desired location
  3. Run glint3d.exe to start the application
  4. Optionally, add to PATH for command-line access
# Add to PATH (PowerShell as Administrator)
$env:PATH += ";C:\path\to\glint3d"
# Verify installation
glint3d --version

macOS Installation

  1. Download the macOS DMG file
  2. Mount the DMG and drag Glint3D to Applications
  3. Allow the app in Security & Privacy settings if prompted
  4. Launch from Applications or use Terminal
# Add to PATH (Terminal)
echo 'export PATH="/Applications/Glint3D.app/Contents/MacOS:$PATH"' >> ~/.zshrc
source ~/.zshrc
# Verify installation
glint3d --version

Linux Installation

  1. Download the Linux tarball (glint3d-linux-x64.tar.gz)
  2. Extract to /opt or your preferred location
  3. Make the binary executable
  4. Add to PATH for system-wide access
# Extract and install
sudo tar -xzf glint3d-linux-x64.tar.gz -C /opt/
sudo chmod +x /opt/glint3d/glint3d
# Add to PATH
echo 'export PATH="/opt/glint3d:$PATH"' >> ~/.bashrc
source ~/.bashrc
# Verify installation
glint3d --version
